# Task API
# Descricao
API REST para gerenciamento de tarefas, construidas com FASTAPI e Python. Um projeto de estudos para aprender desenvolvimento backend.

# Funcionalidades
API REST que permite adicionar itens a uma lista de tarefas, atualizar, listar e deletar (CRUD) tarefas do gerenciador.

# Tecnologias
Python
FastAPI
Uvicorn
Git
SQLite
 
# Como Rodar:
1- Clone o repositorio
git clone https://github.com/pedr0cs/task_api.git
2- Ative o venv
.\\venv\\Scripts\\Activate
3- Instale as dependencias
pip install -r requirements.txt
4- Rode o servidor
uvicorn main:app --reload 

# Documentação
Acesse http://localhost:8000/docs para ver a documentação interativa
